# Results directory guidance

This directory is a placeholder for repository-level organization of generated artifacts.

## Recommended structure

- `results/scrna/`
- `results/bulk-rna/`
- `results/microbiome/`
- `results/spatial/`

## Guidance

- Keep raw intermediate files close to the workflow that generated them.
- Copy final manuscript figures and summary tables into this directory when assembling a release or submission package.
- Record provenance for any manually curated result files in the relevant workflow documentation.
